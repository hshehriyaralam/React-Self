

*** State ***
State is a way to "Store and Manage" data that can change over time and effect how the components renders. We define state using the useState Hook,  which allows you to set an initial value and provides a way to update that state.


*** Hooks ***
Hooks ara a new addition in React 16.8 . They let you see state and other React features without writing a classs . 


*** useState ***
useState Hook allows us to "track state" in a functional components. State generally refers to data or properties that need to be tracking in an applications.
